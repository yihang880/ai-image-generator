
# ai-image-generator

An AI-powered image generation and processing tool using Python. This project demonstrates integration with AI models (e.g., DALL-E) to create images from text prompts and includes basic image manipulation capabilities.

## Features
- Generate images from textual descriptions.
- Basic image processing (e.g., resizing, cropping - *future feature*).

## Technologies
- Python
- OpenAI API (DALL-E 3)
- Pillow (PIL Fork)

## Setup
1. Clone the repository:
   `git clone https://github.com/yihang880/ai-image-generator.git`
2. Install dependencies:
   `pip install openai Pillow`
3. Set your OpenAI API key as an environment variable:
   `export OPENAI_API_KEY='your_api_key_here'`

## Usage
```bash
python main.py
```
